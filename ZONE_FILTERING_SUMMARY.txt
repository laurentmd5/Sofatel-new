🔐 PRINCIPE DE SÉCURITÉ
═══════════════════════════════════════════════════════════════════════════════

  Chaque produit appartient à un emplacement (Emplacement) qui appartient
  à une zone géographique (Zone). Un magasinier ne peut accéder que aux
  produits/mouvements dont:

    Emplacement.zone_id == User.zone_id

  Cette vérification est appliquée à:
    ✓ Tous les GET (lecture données)
    ✓ Tous les POST (création/modification données)
    ✓ Toutes les statistiques et exports

  Les gestionnaires voient TOUS les produits (pas de filtre).

   Rôle: Vérifier qu'un magasinier a une zone assignée

     if user.role == 'magasinier' and user.zone_id is None:

     from zone_rbac import validate_magasinier_zone_access

   Levées:
     400 Bad Request si zone_id manquante


B. validate_emplacement_zone(emplacement_id)
   ──────────────────────────────────────────
   Rôle: Vérifier qu'un emplacement appartient à la zone du magasinier

   Logique:
     emplacement = db.session.get(EmplacementStock, emplacement_id)
     if not emplacement or emplacement.zone_id != current_user.zone_id:
         abort(403, 'Accès refusé: emplacement d\'une autre zone')

   Usage:
     from zone_rbac import validate_emplacement_zone
     validate_emplacement_zone(form.emplacement_id.data)

   Levées:
     403 Forbidden si zone mismatch


C. filter_produit_by_emplacement_zone(query, current_user)
   ────────────────────────────────────────────────────────
   Rôle: Filtrer une query SQLAlchemy de produits par zone

   Logique (simplifié):
     if user_has_global_access(current_user):
         return query  # Chef, gestionnaire → pas de filtre
     if not current_user.zone_id:
         return query.filter(False)  # Magasinier sans zone → aucun résultat
     # Magasinier avec zone:
     return query.join(Emplacement).join(Zone).filter(
         Zone.id == current_user.zone_id
     )

   Usage:
     from rbac_stock import filter_produits_by_zone
     query = Produit.query
     query = filter_produits_by_zone(query, current_user)
     produits = query.all()

   Filtre appliqué:
     Produit.emplacement.zone.id == current_user.zone_id
